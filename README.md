#  简介

一个海量即时通讯系统，使用go net tcp 开发

数据库用到了redis的hash

传输数据为json序列化后的字符

具备了客户端以及服务器端

# 开发环境

| language          | go version go1.16.4            |
| ----------------- | ------------------------------ |
| os                | archlinux/amd64 5.12.8-arch1-1 |
| development tools | vscode                         |
| sql               | redis                          |
